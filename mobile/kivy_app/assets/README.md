# Assets Folder

This folder contains realistic stock images for the home screen services.

## Current Images (Downloaded from Unsplash):

1. **students.png** - Students studying in library
2. **longterm.png** - Modern house exterior
3. **shortterm.png** - Hotel room interior
4. **realestate.png** - House for sale
5. **resorts.png** - Beach resort/vacation destination
6. **shops.png** - Retail shop interior

## Image Specifications:

- **Format:** PNG
- **Size:** 600x400 pixels
- **Quality:** High resolution optimized with Lanczos resampling
- **Source:** Unsplash (free to use)

## Customization:

To replace with your own images, simply:
1. Prepare images at 600x400px (or they'll be auto-resized)
2. Save as PNG format
3. Use the exact filenames above
4. Replace the files in this directory

Run `python generate_images.py` in the parent directory to re-download images or update the Unsplash URLs in that script to use different stock photos.
